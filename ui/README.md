## KIN.ui.page_loader
Displays a standardised loader

#### Methods
* `show` : Displays the loader
* `destroy` : Removs the loader

#### Example
```javascript
  KIN.ui.page_loader.show({text:"Your loading text"})
  
  /* Do your stuff */ 
  
  KIN.ui.page_loader.destroy()  
```
